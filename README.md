# Responsive Image Gallery (CodeAlpha Internship Project)

## 📌 Overview
This project is part of my **Frontend Developer Internship at CodeAlpha**.  
It is a **responsive image gallery** built using **HTML, CSS, and JavaScript**, featuring:
- Pinterest-style masonry layout
- Category filters (All, Nature, City, Animals)
- Lightbox view with next/prev navigation
- Hover effects and smooth transitions
- Fully responsive design for different screen sizes

---

## 🚀 Features
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile.
- **Masonry Grid**: Pinterest-style column layout for images of varying sizes.
- **Interactive Lightbox**: Click an image to view it in full size with navigation controls.
- **Category Filters**: Show/hide images based on category.
- **Hover Effects**: Smooth scale and brightness transitions on hover.

---

## 🛠️ Tech Stack
- **HTML5** – Semantic structure
- **CSS3** – Styling, transitions, responsive design
- **JavaScript (ES6)** – Interactivity, lightbox, filters
